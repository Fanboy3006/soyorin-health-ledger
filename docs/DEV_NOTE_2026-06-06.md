## 架构调整：放弃 Tauri，改用 Webhook + 快捷指令

### 背景
原先计划用 Tauri 打包 iOS App 来接入 Apple Health，但评估后发现：
- Tauri 需要 Rust + Swift + Xcode 签名配置，成本高
- 快捷指令可以直接读取 Apple Health 数据，通过 Webhook 发送到 Supabase

### 新方案
```
iPhone 快捷指令 → 读取 Apple Health → POST Webhook → Supabase Edge Function → biometrics 表 → Soyorin Web App
```

- iPhone 快捷指令读取 Apple Health 数据（体重、血压、心率、步数等）
- 调用 Supabase Edge Function `health-webhook`
- Edge Function 验证 Token 后写入 `biometrics` 表
- 数据自动同步到 Soyorin Web App

### 数据接入方式
```
┌─────────────────────────────────────────────────────────┐
│                    Soyorin 数据流                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  手动录入 ──→ IndexedDB ──→ SyncEngine ──→ Supabase     │
│                                                         │
│  AI 识别 ──→ Edge Function ──→ IndexedDB ──→ Sync       │
│                                                         │
│  快捷指令 ──→ Webhook ──→ Edge Function ──→ Supabase    │
│                    (health-webhook)        (biometrics) │
│                                                         │
│  Supabase ──→ SyncEngine ──→ IndexedDB (下行同步)       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
