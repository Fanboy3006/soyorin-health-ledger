# 开发日志 — 2026-06-07

## 今日完成

### 1. Health Webhook Edge Function

创建 `supabase/functions/health-webhook/index.ts`，用于接收 iPhone 快捷指令发送的 Apple Health 数据。

**功能：**
- 接收 POST 请求，解析 email + date + 体征数据
- 通过 RPC `get_user_id_by_email` 从 `auth.users` 查找 user_id
- 写入 `biometrics` 表（upsert by date + user_id）
- 全局 try/catch + requestId 日志，方便调试
- CORS 支持（OPTIONS 预检）

**文件：**
- `supabase/functions/health-webhook/index.ts` — Edge Function 代码
- `deploy_health_webhook.ps1` — 部署脚本
