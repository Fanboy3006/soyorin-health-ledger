## 待解决问题

### 1. 预设删除后恢复
- **状态**：已修复
- **原因**：`handleDelete` 只删除了本地 IndexedDB，未删除云端 Supabase，同步时又从云端拉回
- **修复**：改用 `deletePresetCompletely()` 同时删除本地和云端

### 2. 批量删除出现双倍内容
- **状态**：待排查
- **可能原因**：
  - 同步逻辑中 `pullTable` 在拉取时未正确去重
  - 或 UI 渲染时 `key` 重复导致 React 渲染异常
  - 或 `pushTable` 在插入后未正确标记 `synced: true`

### 3. `preset_id` 数据库类型

- **SQL**：`ALTER TABLE ledger_entries ALTER COLUMN preset_id TYPE TEXT USING preset_id::TEXT;`
- **执行方式**：通过 Supabase Dashboard SQL 编辑器运行
